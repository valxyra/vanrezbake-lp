# 🚀 Vanrez Bakery Landing Page

## 📖 Overview

The Vanrez Bakery Landing Page is a fast, responsive, and visually appealing web presence built to showcase the Vanrez Bakery brand or product. Leveraging the power of Astro for optimized performance, React for interactive UI components (via Astro Islands), and Tailwind CSS for utility-first styling, this project delivers a modern user experience. It integrates Shadcn UI components, providing a set of beautifully designed and accessible UI primitives, ensuring a consistent and polished look and feel. This repository serves as a boilerplate for a static or highly dynamic landing page, emphasizing speed, developer experience, and design flexibility.

---

## 🛠️ Tech Stack

- **Astro**: For static site generation and performance optimization.
- **React**: For interactive UI components.
- **Tailwind CSS**: For utility-first styling.
- **Shadcn UI**: For accessible UI components.
